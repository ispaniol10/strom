⚙ Supabase Verbindung

Führe folgende SQL-Befehle einmalig in deinem Supabase SQL-Editor aus:

CREATE TABLE IF NOT EXISTS verbrauch_monat ( id BIGSERIAL PRIMARY KEY, jahr INTEGER NOT NULL, monat INTEGER NOT NULL CHECK (monat BETWEEN 1 AND 12), strom_anfang NUMERIC, strom_ende NUMERIC, strom_verbrauch NUMERIC, strom_durchschnitt NUMERIC, strom_eingespeist NUMERIC, gas_anfang NUMERIC, gas_ende NUMERIC, gas_verbrauch NUMERIC, gas_durchschnitt NUMERIC, UNIQUE(jahr, monat) ); CREATE TABLE IF NOT EXISTS einspeisung_solar ( id BIGSERIAL PRIMARY KEY, jahr INTEGER NOT NULL, monat INTEGER NOT NULL CHECK (monat BETWEEN 1 AND 12), zaehler_anfang NUMERIC, zaehler_ende NUMERIC, verguetung NUMERIC, UNIQUE(jahr, monat) );

Optional: Aktiviere Row Level Security und setze Policies für öffentlichen Lesezugriff,
oder nutze den Service-Key für vollen Schreibzugriff (nur lokal verwenden!).
